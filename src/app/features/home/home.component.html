Europa Component Library

The Europa Component Library (ECL) is a comprehensive style guide containing the design elements and visual standards that make up all Europa websites and applications.

This library contains all available components that you can use to build your own website.

All library elements are accompanied by

  • documentation explaining what a component is and how it should be used
  • a visual demonstration
  • HTML/CSS code for implementation

eUI/ECL integration

eUI provides Angular implementation of ECL components, enabling projects based on angular, get all benefints of having ECL compliant web application.

It is also possible to use all eUI components in eui/ECL application by importing specific component module or by importing all components pulling UxAllModule

Documentation

Documentation can be found as live showcase of all components available, as well as other building blocks, such as utility css classes, typography, colors etc.
